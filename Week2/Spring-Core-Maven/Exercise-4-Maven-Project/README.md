# Maven Project Configuration

## Objective

Create and configure a Maven-based Java project using Spring Framework.

## Technologies

- Java 17
- Maven
- Spring Core

## Project Structure

```
src
 ├── main
 │     ├── java
 │     └── resources
 └── test
```

## Features

- Maven Project Setup
- Dependency Management
- Java Packaging
- Build Automation

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Spring-Core-Maven.git
```

2. Import as a Maven Project in IntelliJ IDEA or Eclipse.

3. Run:

```
App.java
```
