# Exercise 2 - Dependency Injection

## Objective

Demonstrate Dependency Injection (DI) using Spring Framework.

## Concepts Covered

- Inversion of Control (IoC)
- Setter Injection
- Spring Bean Configuration
- XML Configuration
- Loose Coupling

## Project Structure

```text
repository/
service/
model/
Main.java
applicationContext.xml
```

## Expected Output

```
Welcome to Spring Dependency Injection!
```

## Technologies Used

- Java
- Spring Core
- Maven

## Learning Outcomes

- Understand Spring IoC Container
- Configure Beans using XML
- Implement Setter Injection
- Build loosely coupled applications
