# HOL 01 – My First React App

## Objective
Set up a React environment and create a first React application that displays a welcome heading.

## What it does
- Creates a React application named `myfirstreact`
- Displays the heading: **"welcome to the first session of React"**

## How to Run
```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Key Concepts
- Setting up a React environment
- Using `create-react-app`
- JSX rendering
- ReactDOM.render()
